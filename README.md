# Roupang 

-------


🐈 프로젝트 설명 (Description)

고양이 용품을 판매하는 집사들을 위한 쇼핑몰 루팡입니다.

<br>

🐈 프로젝트 목표 (Project goal)

1.	Spring Boot와 React의 기술적 이해도를 높이고, 이를 활용한 쇼핑몰 개발 경험을 쌓는다.
2.	쇼핑몰의 기본 기능을 구현하고, 이를 사용자 친화적으로 디자인한다.
3.	쇼핑몰의 성능과 보안을 개선한다.
